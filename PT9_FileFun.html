File handling in C

File Handling is the is the storing of data in a file using a program. In file handling, some function are used to handle the file.

Functions of File Handling

  1. fopen()
    2. It is used to opens new or already created file.
  2. fclose()
    3. It is used to closes the file.
  3. fprintf()
    4. It is used to write data into the file.
  4. fputc()
    5. It is uesd to writes a character into the file.
  5. fputw()
    6. It is uesd to writes an integer to file.
  6. fscanf()
    7. It is used to reads data from the file.
  7. fgetc()
    8. It is used to reads a character from file.
  8. fgetw()
    9. It is used to reads an integer from file.
  9. fseek()
    10. It is used to sets the file pointer to given position.
  10. ftell()
    11. This function is used to returns current position.
  11. rewind()
    12. It is used to sets the file pointer to the beginning of the file.
